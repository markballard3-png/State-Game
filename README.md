# Capital Kickoff: NCAA State Stadium Challenge

A desktop-first Next.js learning game for private family use that teaches all 50 U.S. states, their locations, and their capitals through a polished college-football theme.

## Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- Local storage for saved progress

## Features

- Practice Facility, Road Trip Season, Rivalry Week, Bowl Review, Championship Mode
- Parent Dashboard with mastery, misses, accuracy, and recommended practice
- Full 50-state data set with capitals, regions, memory hooks, and football anchors
- Desktop dashboard layout with scoreboard, recruiting board, and state cards
- Local-only progress saving for private home use

## Development

1. Install dependencies:

```bash
npm install
```

2. Start the app:

```bash
npm run dev
```

3. Open `http://localhost:3000`

## Deployment

This project is structured for Vercel deployment with no backend required.

## Assets

You can optionally place official or custom team graphics in `public/logos`. The app is already set up to work without them by using helmet-style placeholders and team colors.
