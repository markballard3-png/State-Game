# State Game

A lightweight browser game for practicing U.S. states and capitals.

## Project structure

- `index.html` boots the app.
- `src/main.js` contains the game logic and rendering.
- `src/data/states.js` holds the starter data set.
- `src/styles.css` defines the visual design.

## Getting started

Because this project is dependency-free, you can open `index.html` directly in a browser or serve the folder with any static file server.
